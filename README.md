# 🚀 Compresor Elite Bot
Bot profesional para comprimir videos con **FFmpeg CRF** y **Cola de Espera**.

### 🌟 Funciones
- **60GB Optimizados:** Limpieza automática de disco.
- **Auto-Split:** Divide videos de >2GB automáticamente.
- **Configuración Estándar:** Listo para usar sin tocar nada.
- **Emojis Profesionales:** Interfaz visual clara.

### 🛠️ Despliegue
1. Sube estos archivos a GitHub.
2. Configura las variables de entorno.
3. El `Dockerfile` hará el resto.
